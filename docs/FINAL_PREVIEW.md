## 🚀 Deploy in 3 Steps

### **Step 1: Push to GitHub**
```bash
cd your-logicore-repo
git add docs/
git commit -m "feat(docs): add complete documentation with icon-based sidebar"
git push origin main
```

### **Step 2: Enable GitHub Pages**
1. Go to your repo → **Settings**
2. Click **Pages** on left
3. Select **Branch**: `main` | **Folder**: `/docs`
4. Click **Save**

### **Step 3: Done!** ✅
Your docs are live at: `https://yourusername.github.io/logicore`

## 💡 Pro Tips

### Customize Icons
Don't like the icons? Edit `_sidebar.md` and change them:

```markdown
# Change 🤖 to 🧠 for Agents
* 🧠 [Agents](...)

# Change 💾 to 📊 for Memory
* 📊 [Memory](...)
```

### Change Theme Color
Edit `index.html` line 120:
```javascript
themeColor: "#673ab7",  // Purple
// Change to:
themeColor: "#2196F3",  // Blue
themeColor: "#FF5722",  // Orange
```
